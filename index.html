About

I began creating my first application through self-learning from the internet, using ReactJS and Laravel. I felt it was truly magical, creating a system and its interface with integrated code. Here, I learned frontend/client-side development using ReactJS and Redux, and backend/server-side development with ExpressJS, MongoDB, and MySQL. Additionally, I honed my styling skills using Tailwind and Bootstrap. I am motivated to continue learning programming not only because I enjoy it, but also because programming languages, libraries, frameworks, and related technologies continue to evolve over time.
